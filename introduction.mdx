# Welcome to DadoMatch Shared

DadoMatch Shared is a Kotlin Multiplatform library providing shared business logic, UI components, and utilities for DadoMatch iOS and Android applications.

## Platform Support

- **Android**: API 24+ (Android 7.0 Nougat and above)
- **iOS**: iOS 16.0+ with XCFramework distribution

## Architecture

DadoMatch Shared follows clean architecture principles with feature-based organization:

- **Core Module**: Shared utilities, configuration, and base classes
- **Feature Modules**: Authentication, Icebreaker, Subscription, Onboarding, Success
- **Platform-Specific Code**: Android and iOS implementations via `expect`/`actual`

## What's Inside

The library provides:

- **Authentication**: Firebase auth with Google, Apple, and anonymous sign-in
- **Icebreaker Generation**: AI-powered conversation starters using Gemini
- **Subscription Management**: RevenueCat SDK integration for Pro features
- **UI Components**: Reusable Compose Multiplatform screens and components
- **Data Layer**: Repository pattern with Room database persistence
- **Environment Configuration**: BuildKonfig for environment-specific settings
