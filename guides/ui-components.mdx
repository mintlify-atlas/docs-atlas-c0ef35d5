## Selector Components

### SelectorGroup

A horizontal selector for environment and intensity options:

```kotlin
@Composable
fun SelectorGroup(
    title: String,
    options: List<String>,
    selectedOption: String,
    onOptionSelected: (String) -> Unit,
    selectionColorProvider: (String) -> Color,
    iconProvider: (String) -> String,
    isRestricted: ((String) -> Boolean)? = null
)
```

**Usage from HomeScreen:**

```kotlin
// Environment Selector
SelectorGroup(
    title = "Environment",
    options = AppConstants.ENVIRONMENTS,
    selectedOption = selectedEnvironment,
    onOptionSelected = { selectedEnvironment = it },
    selectionColorProvider = { AppConstants.getEnvironmentColor(it) },
    iconProvider = { AppConstants.getEnvironmentIcon(it) }
)

// Intensity Selector with premium restrictions
SelectorGroup(
    title = "Intensity",
    options = AppConstants.INTENSITIES,
    selectedOption = selectedIntensity,
    onOptionSelected = { selectedIntensity = it },
    selectionColorProvider = { AppConstants.getIntensityColor(it) },
    iconProvider = { AppConstants.getIntensityIcon(it) },
    isRestricted = { option -> option == "int_spicy" && !uiState.isPremium }
)
```

**Features:**

- Horizontal scrolling for multiple options
- Custom color per option
- Icon support
- Premium restriction badges
- Smooth selection animations

<Warning>
  Always provide the `isRestricted` callback to lock premium content for free users. This provides visual feedback before the user attempts to use restricted features.
</Warning>

## Bottom Sheet Components

### AuthBottomSheet

A complete authentication interface with glassmorphic design:

```kotlin
@Composable
fun AuthBottomSheet(
    viewModel: AuthViewModel,
    onDismiss: () -> Unit
)
```

**Full Implementation:**

```kotlin
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun MyScreen() {
    val authViewModel: AuthViewModel = koinViewModel()
    val authUiState by authViewModel.uiState.collectAsState()
    var showAuth by remember { mutableStateOf(false) }
    
    val sheetState = rememberModalBottomSheetState(
        confirmValueChange = { targetValue ->
            // Prevent swipe-to-dismiss while sign-in is in progress
            if (authUiState.isLoading && targetValue == SheetValue.Hidden) false
            else true
        }
    )
    
    if (showAuth) {
        ModalBottomSheet(
            onDismissRequest = { 
                if (!authUiState.isLoading) showAuth = false 
            },
            sheetState = sheetState,
            containerColor = Color.Transparent,
            scrimColor = Color.Black.copy(alpha = 0.5f)
        ) {
            AuthBottomSheet(
                viewModel = authViewModel,
                onDismiss = { showAuth = false }
            )
        }
    }
}
```

**Features:**

- Glassmorphic container with gradient background
- Prevents dismissal during authentication
- Auto-dismisses on successful sign-in
- Loading state with circular progress
- Error display with retry option
- Terms and conditions text

## Styling Best Practices

<Steps>
  <Step title="Use Theme Colors">
    Always use theme colors for consistency:

    ```kotlin
    Text(
        text = "Welcome",
        color = TextWhite,  // Not Color.White
        style = MaterialTheme.typography.headlineMedium
    )
    ```
  </Step>

  <Step title="Apply Consistent Shapes">
    Use consistent corner radius:

    ```kotlin
    Box(
        modifier = Modifier
            .clip(RoundedCornerShape(16.dp))  // Standard radius
            .background(DarkSurface)
    )
    ```
  </Step>

  <Step title="Implement Glassmorphism">
    Create depth with gradient backgrounds and borders:

    ```kotlin
    Box(
        modifier = Modifier
            .background(
                Brush.verticalGradient(
                    colors = listOf(
                        Color(0xFF1E1E2C).copy(alpha = 0.9f),
                        DeepDarkBlue.copy(alpha = 0.98f)
                    )
                )
            )
            .border(
                width = 1.dp,
                brush = Brush.verticalGradient(
                    colors = listOf(
                        Color.White.copy(alpha = 0.3f),
                        Color.Transparent
                    )
                )
            )
    )
    ```
  </Step>

  <Step title="Add Shadow Effects">
    Use shadow modifiers for depth:

    ```kotlin
    Button(
        modifier = Modifier.shadow(
            elevation = 20.dp,
            shape = RoundedCornerShape(32.dp),
            ambientColor = NeonPink,
            spotColor = NeonCyan
        )
    )
    ```
  </Step>
</Steps>
